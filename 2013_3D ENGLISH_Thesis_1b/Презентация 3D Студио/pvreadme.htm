О средстве просмотра PowerPoint

Средство просмотра Microsoft Office PowerPoint позволяет просматривать полнофункциональные презентации, созданные в Microsoft Office PowerPoint 2003. Средство просмотра PowerPoint позволяет просматривать презентации, созданные в PowerPoint 97, PowerPoint 2000 и PowerPoint 2002. Средство просмотра PowerPoint также поддерживает открытие презентаций PowerPoint, защищенных паролем.

Системные требования

  • ПК с процессором с тактовой частотой 200 МГц или выше
  • Microsoft Internet Explorer 5 или более поздней версии
  • Поддерживаемые операционные системы: Microsoft Windows 98 Second Edition, Windows Millennium Edition, Windows 2000 с пакетом обновления 3 (SP3), Windows XP или Windows Server 2003.
  • ОЗУ:
    • Для Windows 98 SE, Windows ME: 32 Мбайт ОЗУ
    • Для Windows 2000 с пакетом обновления 3 (SP3): 64 Мбайт ОЗУ
    • Для Windows XP, Windows Server 2003: 128 Мбайт ОЗУ
  • 5 Мбайт свободного места на жестком диске
  • Монитор Super VGA (800 x 600) или с более высоким разрешением

Многоязыковая поддержка

Средство просмотра PowerPoint не поддерживает средства проверки орфографии Microsoft Office.

Средство просмотра PowerPoint не поддерживает презентации, созданные на двух языках в PowerPoint 97.

Возврат к содержанию

Работа со средством просмотра PowerPoint

Средство просмотра PowerPoint неправильно показывает анимацию схемы. Вместо показа анимации схема будет показана завершенной.

Средство просмотра PowerPoint не поддерживает презентации средства управления правами на доступ к данным (IRM).

Средство просмотра PowerPoint не поддерживает запуск макросов, программ и открытие связанных или внедренных объектов.

Средство просмотра PowerPoint не поддерживает отображение связанных с презентацией файлов следующих графических форматов: EPS, PCT, EMZ, CDR, CGM, WPG и WMZ.
